# config_checklist.md

Manually check config files for the following:

* `config.yaml`
    * Downloaded from GitHub, correct branch
    * Run name has been set
    * Changed any scientific parameters that are different from default

* `ex_lanes.csv`
    * Cross reference with experimental plan, check:
        * R1 filename
        * R2 filename

* `ex_samples.csv`
    * Cross reference with experimental plan, check:
        * SeqID
        * Quadruplex
        * DonorID
        * Age
        * Sex

* `ex_technical_controls.csv`
    * Cross reference with experimental plan, check:
        * SeqID
        * Quadruplex
    
* `ms_samples.csv`
    * Cross reference with experimental plan, check:
        * SeqID
        * R1 filename
        * R2 filename
        * DonorID
        * Age
        * Sex
