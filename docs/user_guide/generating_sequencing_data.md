# Generating sequincing data

* Each EX and MS sample pair must be from the same individual. 
* The MS sample does not have to be the same tissue as the EX sample. 
* A single MS sample can be used for multiple EX samples from the same individual.

### EX samples

Use `Basic Protocol 1` from [Phie et al. 2026](doi)

### MS samples

Use `Basic Protocol 2` from [Phie et al. 2026](doi)
